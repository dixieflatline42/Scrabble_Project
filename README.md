# Sistema de Jogo de Scrabble

Este projeto implementa um **sistema completo de jogo de Scrabble** desenvolvido em **C++**, com foco em **lógica de programação orientada a objetos** e **boas práticas de design**.

O objetivo é oferecer uma base sólida para estudos de **POO**, **estruturas de dados** e **arquitetura modular**, simulando o comportamento real de um Scrabble.

---

## Principais Funcionalidades

- Regras parciais o jogo Scrabble 
- Representação visual do tabuleiro diretamente no console
- Alternância automática entre os jogadores 
- Validação de jogadas inválidas
- Estrutura modular para futuras expansões (como interface gráfica)
